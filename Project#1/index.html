HTML5 is the latest standard that defines HTML. It comes with a number of new elements, attributes, and behaviours. It provides new semantics, graphics, and multimedia elements. HTML5 is designed to deliver rich web content without the need for additional plug-ins. HTML5 ultimately contains a larger set of technologies that allows more diverse and powerful sites and applications on the Web.

Here is a standard structure of an HTML5 document:

<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8" />
   <title>WEB222</title>
   <link href="css/mystyle.css" rel="stylesheet"/>
   <script src="js/myscript.js"></script>
</head>
<body>
   <h1>Basic HTML Document Structure</h1>
   <p>This is a paragraph.</p>
   <p>Here are links to
      <a href="https://ict.senecacollege.ca/">School of ICT</a> and
      <a href="http://www.senecacollege.ca/">Seneca College.</a>    </p>
</body>
</html>
